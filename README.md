## Overview

A social feed application built during the **Rocketseat Ignite** program to master React fundamentals. Covers components, props, state management, hooks, and event handling.

## Features

- Post creation with author and content
- Comment system with like/delete functionality
- Avatar and author info components
- Form validation and controlled inputs
- CSS Modules for scoped styling

## Tech Stack

| Technology | Purpose |
|---|---|
| **React** | UI library |
| **Vite** | Build tool |
| **JavaScript** | Core language |
| **CSS Modules** | Scoped styling |

## Getting Started

```bash
git clone https://github.com/rafaumeu/fundamentos-reactjs.git
cd fundamentos-reactjs
npm install
npm run dev
```
